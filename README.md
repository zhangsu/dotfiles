## Installation

```Zsh
git clone --recursive git@github.com:zhangsu/dotfiles.git
cd dotfiles
rake install_oh_my_zsh  # this also changes the shell
rake
rake install_ruby_build # optional for rbenv
```

## Update submodules

```Zsh
git submodule update --init --recursive --remote
```
